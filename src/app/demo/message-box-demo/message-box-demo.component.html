Message Box Component Demo

Informational Messages

This is an information message with a blue background and border. It can contain any HTML content within the message body.

Warning Messages

This is a warning message with a yellow background and border. This type could be used for alerts about incomplete information or potential issues.

Error Messages

This is an error message with a red background and border. Use this when something has gone wrong or critical information is missing.

Success Messages

This is a success message with a green background and border. This could be used to confirm that all requirements have been met, such as Ready for Final Decision status.
